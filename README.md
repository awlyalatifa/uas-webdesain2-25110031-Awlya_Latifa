**Deskripsi:**
Website ini merupakan Company Profile untuk UMKM kuliner **Yummi Company Bakery Pekanbaru**, sebuah usaha bakery yang memproduksi roti, pastry, dan cake premium dengan standar higienis dan bahan baku pilihan. Website dibuat sebagai tugas UAS mata kuliah Web Desain 2, bertujuan mempromosikan katalog produk unggulan (roti manis, cakes & tart, pastry), menonjolkan standarisasi kualitas dan kebersihan dapur, serta mempermudah pelanggan memesan produk maupun kue custom untuk acara khusus secara online.

## 🍞 Halaman
- **Beranda** — Hero section dengan tagline utama, badge jam operasional, dan slider carousel menu terlaris mingguan (Butter Croissant).
- **Katalog Menu** — Daftar lengkap produk berdasarkan kategori Roti Manis, Cakes & Tart, dan Pastry, lengkap dengan foto, deskripsi, dan harga per item/loyang.
- **Tentang Kami** — Identitas dan filosofi brand (bahan baku pilihan, higienitas terjamin, panggang setiap pagi), informasi fasilitas outlet, serta layanan pemesanan kue custom untuk acara khusus.
- **Kontak & Lokasi** — Formulir digital kontak/pemesanan, alamat outlet resmi, tautan WhatsApp admin, akun Instagram, serta embed Google Maps lokasi toko.

## ⚙️ Fitur JavaScript
- Bootstrap Carousel untuk slider menu terlaris mingguan.
- Modal "Detail Produk" untuk menampilkan deskripsi lengkap tiap item menu.
- Tombol pemesanan langsung per produk yang terhubung ke API WhatsApp admin.
- Validasi form kontak/pemesanan sebelum data berhasil terkirim.
